# エンディング到達可能性チェック

## エンディング条件一覧

1. **END 1**: E>=5 && S>=3 && T>=1
2. **END 2**: T>=1 && S>=4 && E>=1
3. **END 3**: T>=2 && S>=3 && E<4
4. **END 4**: E>=4 && T<=-1
5. **END 5**: S>=5 && E<3
6. **END 6**: T>=3 && E>=2 && S<=2
7. **END 7**: S>=6
8. **規定到達**: シーン17個見たらEND 6（デフォルト）

## 各エンディングへの到達可能性分析

### END 1: E>=5 && S>=3 && T>=1
**到達可能**: ✅
- EVIDを+1する選択肢が多数存在
- SHADOWを+1以上する選択肢が多数存在
- TRUSTを+1する選択肢が多数存在
- 例: シーン1「ポケットにしまう」(E+1), シーン2「音の方へ行く」(S+1), シーン3「紙を読む」(E+1,S+1) など

### END 2: T>=1 && S>=4 && E>=1
**到達可能**: ✅
- TRUST+1の選択肢が多数
- SHADOW+1以上の選択肢が多数
- EVID+1の選択肢が多数
- 例: シーン1「磨いて棚に戻す」(T+1), シーン2「音の方へ行く」(S+1), シーン3「紙を読む」(E+1) など

### END 3: T>=2 && S>=3 && E<4
**到達可能**: ✅
- TRUST+1の選択肢が多数（最低2回必要）
- SHADOW+1以上の選択肢が多数（最低3回必要）
- E<4なので、EVIDを増やしすぎない選択肢を選ぶ必要がある
- 例: シーン1「磨いて棚に戻す」(T+1), シーン2「音の方へ行く」(S+1), シーン4「当主に渡す」(T+1), シーン5「確かめに行く」(S+1), シーン6「外へ出る」(S+1) など

### END 4: E>=4 && T<=-1
**到達可能**: ✅
- EVID+1の選択肢が多数（最低4回必要）
- TRUST-1の選択肢が存在
- 例: シーン1「ポケットにしまう」(E+1,T-1), シーン3「紙を読む」(E+1), シーン4「黙って仕舞う」(E+1,T-1), シーン7「証拠を見せる」(E+1) など

### END 5: S>=5 && E<3
**到達可能**: ✅
- SHADOW+1以上の選択肢が多数（最低5回必要）
- E<3なので、EVIDを増やしすぎない選択肢を選ぶ必要がある
- 例: シーン2「音の方へ行く」(S+1), シーン5「確かめに行く」(S+1), シーン6「外へ出る」(S+1), シーン7「無言で通る」(S+1), シーン10「窓を開ける」(S+1), シーン13「同じ回数だけ叩く」(S+2) など

### END 6: T>=3 && E>=2 && S<=2
**到達可能**: ✅
- TRUST+1の選択肢が多数（最低3回必要）
- EVID+1の選択肢が多数（最低2回必要）
- S<=2なので、SHADOWを増やしすぎない選択肢を選ぶ必要がある
- 例: シーン1「磨いて棚に戻す」(T+1), シーン2「作業に戻る」(T+1), シーン3「捨てる」(T+1), シーン4「当主に渡す」(T+1), シーン7「証拠を見せる」(T+1,E+1) など

### END 7: S>=6
**到達可能**: ✅
- SHADOW+2の選択肢が複数存在（シーン13「同じ回数だけ叩く」、シーン17「隙間から外を覗く」、シーン21「板の裏を覗く」、シーン23「曇りを拭き切って確かめる」、シーン25「ランプの奥を覗く」、シーン28「土を少し掘って確かめる」）
- SHADOW+1の選択肢も多数存在
- 例: シーン2「音の方へ行く」(S+1), シーン5「確かめに行く」(S+1), シーン6「外へ出る」(S+1), シーン13「同じ回数だけ叩く」(S+2), シーン17「隙間から外を覗く」(S+2) など

## 潜在的な問題点

### 1. エンディング判定の優先順位
現在のコードでは、エンディング条件の判定順序が重要です。複数の条件を同時に満たす場合、先に判定されたエンディングが表示されます。

### 2. 規定到達エンディング
`maxScenes:17` に設定されていますが、実際のシーン数は33個以上あります。17シーン見た後にエンディングに到達しない場合、規定到達エンディングが表示されます。

### 3. エンディング到達のタイミング
`endByFormula()`は選択肢選択後とシーン遷移時に呼ばれますが、全ての分岐で確実にエンディングに到達できるかは、プレイヤーの選択次第です。

## 推奨事項

1. **エンディング到達保証**: 全てのエンディングに到達可能であることを確認済み ✅
2. **maxScenesの調整**: 現在17に設定されていますが、実際のシーン数に合わせて調整するか、より多くのシーンを見られるようにすることを検討
3. **エンディング判定の最適化**: より厳しい条件から判定することで、意図したエンディングが表示されるようにする
