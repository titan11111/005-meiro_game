# フィールド背景画像の設定方法

## 概要
フィールドマップを背景画像として表示し、その上に歩行アニメーションを優先表示するように実装しました。

## 背景画像の準備

### 1. フィールドマップのスクリーンショットを撮る

1. ゲームをブラウザで開く
2. フィールドマップ部分を表示
3. ブラウザの開発者ツール（F12）を使用して、`#field-map` 要素を選択
4. スクリーンショットを撮る、または以下の方法で画像を取得：
   - ブラウザの拡張機能を使用
   - 画面キャプチャツールを使用
   - 開発者ツールの「Capture node screenshot」機能を使用

### 2. 画像ファイルの保存

1. スクリーンショットを `field_background.png` という名前で保存
2. `77プロトタイプ１` フォルダに配置（`index.html` と同じフォルダ）

### 3. 画像の推奨サイズ

- 幅: 600px 程度（ゲームコンテナの幅に合わせる）
- 高さ: 200-300px 程度（フィールドマップの高さに合わせる）
- 形式: PNG（透明部分があれば維持）

## 自動設定

現在の実装では、`field_background.png` が存在する場合、自動的に背景として使用されます。
画像が見つからない場合は、デフォルトの背景色が使用されます。

## カスタマイズ

背景画像のパスを変更する場合は、`setFieldBackground()` メソッド内の以下を変更：

```javascript
const backgroundImage = './field_background.png'; // ここを変更
```

## 歩行アニメーションの優先表示

- 歩行アニメーションは `z-index: 100` で最前面に表示されます
- 背景画像の上に重ねて表示されます
- セルの絵文字は非表示になり、背景画像と歩行アニメーションのみが表示されます

## トラブルシューティング

### 背景画像が表示されない
1. ファイル名が `field_background.png` であることを確認
2. `index.html` と同じフォルダに配置されているか確認
3. ブラウザのコンソールでエラーを確認

### 歩行アニメーションが表示されない
1. スプライトシートが正しく読み込まれているか確認（コンソールで確認）
2. スプライトの設定（サイズ、フレーム数）が正しいか確認
3. ブラウザのコンソールでエラーを確認
