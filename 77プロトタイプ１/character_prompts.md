## キャラクター設定の詳細

### 外見的特徴
- **年齢**: 50代前半〜中盤
- **体型**: 普通体型、やや疲れた様子
- **服装**: カジュアルな日常着（ポロシャツ、チノパン、スニーカー）
- **持ち物**: 木の棒（武器として使用）
- **表情**: 優しく、少し疲れているが粘り強い

### 性格・雰囲気
- 親しみやすく、共感できる
- 日常的な存在感
- 決意に満ちた、でも無理をしない
- 現実的で等身大

### 色の提案
- **服**: 薄い青、ベージュ、グレー系（落ち着いた色）
- **肌**: 健康的な肌色
- **全体的なトーン**: 温かみのある、柔らかい色合い

## ネガティブプロンプト（避けるべき要素）

```
avoid: muscular, heroic pose, fantasy armor, exaggerated features, too young, too energetic, anime style with big eyes, overly detailed, dark atmosphere, scary, aggressive
```

**日本語版**:
```
避ける: 筋肉質、英雄的なポーズ、ファンタジーの鎧、誇張された特徴、若すぎる、エネルギッシュすぎる、大きな目のアニメ風、過度に詳細、暗い雰囲気、怖い、攻撃的
```

## 生成時の注意点

1. **サイズ**: 用途に応じて適切なサイズを指定（32x32、64x64、128x128、512x512など）
2. **背景**: 透明背景またはシンプルな背景を推奨
3. **表情**: 優しく、親しみやすい表情を重視
4. **服装**: 現実的で日常的な服装を維持
5. **年齢感**: 50代の落ち着いた雰囲気を保つ
