# 家の見取り図 3D - フローチャート

## 部屋の接続関係

```
                    [玄関] (スタート地点)
                      │
                      │ 東
                      ↓
                    [廊下]
            ┌─────────┼─────────┐
            │ 西      │ 東      │ 南
            │         │         │
          [玄関]    [居間]    [階段]
                      │         │
                      │ 南      │ (上る)
                      ↓         ↓
                    [台所]  [二階廊下]
                      │         │
                      │ 西      ├─────────┐
                      ↓         │ 東      │ 西
                    [物置]    [寝室]  [子ども部屋]
                      │
                      │ 東
                      ↓
                    [台所] (戻る)
```

## 詳細な部屋マップ

### 1階
- **玄関** (x:0, y:0, z:0) - スタート地点
  - 出口: 東 → 廊下
  - 家具: フックの見取り図、下駄箱、姿見の鏡、古い電話
  - **ゴール地点**: 正しい見取り図を持って戻ると脱出成功

- **廊下** (x:10, y:0, z:0)
  - 出口: 西 → 玄関、東 → 居間、南 → 階段
  - 家具: 額縁、傘立て、掛け時計

- **居間** (x:20, y:0, z:0)
  - 出口: 西 → 廊下、南 → 台所
  - 家具: テレビ、テーブル、ソファ、本棚
  - **重要**: テレビを調べると「赤い部屋」が出現（章2以上）

- **台所** (x:20, y:0, z:10)
  - 出口: 北 → 居間、西 → 物置
  - 家具: 冷蔵庫、流し、包丁立て、食器棚
  - **重要**: 冷蔵庫（不一致度60%以上）→ 「正しい見取り図」を取得

- **物置** (x:10, y:0, z:10)
  - 出口: 東 → 台所
  - 家具: 段ボール、古いラジオ、棚
  - **重要**: 段ボールを調べると章3に進み、家の構造が変わる

- **階段** (x:10, y:0, z:15)
  - 出口: 北 → 廊下
  - 家具: 階段（上る）、手すり
  - **重要**: 階段を調べると2階へ移動

### 2階
- **二階廊下** (x:10, y:5, z:15)
  - 出口: 東 → 寝室、西 → 子ども部屋
  - 家具: 階段（下りる）、花瓶、窓

- **寝室** (x:20, y:5, z:15)
  - 出口: 西 → 二階廊下
  - 家具: ベッド、クローゼット、ランプ

- **子ども部屋** (x:0, y:5, z:15)
  - 出口: 東 → 二階廊下
  - 家具: おもちゃ箱、机、落書きの壁
  - **重要**: 落書きの壁を調べると「赤い部屋」が出現

### 条件付き部屋
- **赤い部屋** (x:30, y:0, z:0) - 出現条件あり
  - 出口: 西 → 居間
  - 出現条件:
    1. 居間のテレビを調べる（章2以上）
    2. 子ども部屋の落書きの壁を調べる
  - 家具: 方眼の床

## ゴールまでのフロー

### 脱出エンド（良いエンド）
```
1. 玄関でスタート
   ↓
2. 廊下 → 居間 → 台所 など、複数の部屋を探索
   ↓
3. 不一致度を上げる（各家具を調べる）
   - 各家具を調べると不一致度が2〜5%上昇
   ↓
4. 不一致度60%以上になる
   ↓
5. 台所の冷蔵庫を調べる
   → 「正しい見取り図」を取得
   ↓
6. 玄関に戻る
   ↓
7. 脱出成功！✨
```

### バッドエンド
```
1. 探索を続ける（12ステップ以上）
   ↓
2. 章4に到達
   ↓
3. 図面の中に閉じ込められる...
```

## 章の進行

- **章1**: 初期状態
- **章2**: 居間のテレビを調べる、または3ステップ以上
- **章3**: 物置の段ボールを調べる、または7ステップ以上
  - この時、家の構造が変わる（reorganize）
- **章4**: 12ステップ以上でバッドエンド

## 重要なアイテム

- **古い見取り図**: 初期所持
- **正しい見取り図**: 台所の冷蔵庫（不一致度60%以上）で取得
  - これを持って玄関に戻ると脱出成功

## 探索のヒント

1. **最低3部屋以上**を探索する必要がある（目標1）
2. **不一致度を上げる**には各家具を調べる
3. **正しい見取り図**は台所の冷蔵庫から（不一致度60%以上が条件）
4. **玄関に戻る**ことが脱出の鍵

## 部屋の探索順序の例

### 最短ルート（脱出エンド）
```
玄関 → 廊下 → 居間 → 台所
  ↓
冷蔵庫を調べる（不一致度60%以上必要）
  ↓
玄関に戻る
```

### 全探索ルート
```
玄関 → 廊下 → 居間 → 台所 → 物置
  ↓
階段 → 二階廊下 → 寝室
  ↓
二階廊下 → 子ども部屋
  ↓
居間 → 赤い部屋（出現後）
  ↓
台所（冷蔵庫で正しい見取り図取得）
  ↓
玄関（脱出）
```

## 注意点

- すべての部屋は接続されているため、どの順序でも探索可能
- 「赤い部屋」は条件を満たさないと出現しない
- 章3で家の構造が変わる（reorganize）が、すべての部屋は依然として到達可能
- 不一致度が上がると、世界が歪み始める（視覚効果が変化）
